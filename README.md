# shena4746/perl-for-latex

Alpine Linux image just containing Perl modules that are required by a latex tool latexindent.
The image is intended to be used as material for multi-stage build.

- Image contains
  - YAML::Tiny
  - Log::Dispatch
  - Unicode::LineBreak
  - Log::Log4perl
  - File::HomeDir
